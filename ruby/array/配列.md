## 配列
<br>

- 変数は1つの中身を入れ替える箱だが、配列は複数の値を保持できる箱  
```rb
names = ["tanaka", "nakamura", "satou"]
puts names
```
<br>

```
#実行結果
tanaka
nakamura                                         
satou   
```
<br>
<br>

- インデックス番号で取り出せる  
```rb
names = ["tanaka", "nakamura", "satou"]
puts names[1]
#=> nakamura
```
<br>
<br>

- 配列に要素を追加する  
```rb
names = ["tanaka", "nakamura", "satou"]
names << "suzuki" #配列の末尾に追加される
p names
#=> ["tanaka", "nakamura", "satou", "suzuki"]
```
