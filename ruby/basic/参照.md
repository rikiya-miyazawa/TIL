## 参照  
<br>

- 参照の概念  
```
Rubyの変数にはオブジェクトそのものではなく、「オブジェクトへの参照」が格納されている
同じオブジェクトを参照しているかどうかはobuject_idメソッドを使うとわかる
```
<br>

```rb
a = 'hello'
b = 'hello'
p a.object_id  #=> 60
p b.object_id  #=> 80

c = b
p c.object_id  #=> 80

def m(d)
  p d.object_id
end
m(c)  #=> 80
m(a)  #=> 60

#同じオブジェクトを参照している場合、オブジェクトの状態が変更されると、その変更がそのまま各変数に影響する


```
