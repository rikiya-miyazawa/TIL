## ブロック  
<br>

- ブロック  
```
ブロックとはメソッドの引数として渡すことができる処理の塊のこと
例えばeachメソッドで取り出した要素をどのように扱うかなどをブロックに記述する
```
<br>

```rb
#do ~ end までがブロック
numbers = [1, 2, 3, 4]
sum = 0
numbers.each do |n|
  sum += n
end
sum  #=> 10
```
<br>

- 配列の要素を削除する条件を自由に指定する  
```rb
#delete_ifメソッドを利用する
#配列から奇数の要素を削除する
#odd?は数値が奇数であればtrueを返す
#delete_ifメソッドでtrueの要素は削除される
a = [1, 2, 3, 1, 2, 3]
a.delete_if do |n|
  n.odd?
end
a  #=> [2, 2]
```
