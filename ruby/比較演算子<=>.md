## 比較演算子<=>について 
<br>

- 比較演算子<=> 
```
2つのオブジェクトを比較し、以下のような結果を返す。

・左辺が右辺より小さい場合は、負の整数(-1)を返す。
・左辺が右辺と等しい場合は、0を返す。
・左辺が右辺より大きい場合は、正の整数(1)を返す。

この演算子は、ソートや検索などで利用される。
例えば、配列の要素を昇順に並べ替える場合は a <=> b と書く。
また、降順に並べ替える場合は -1 * (a <=> b) または b <=> a または -a <=> -b と書く。
```
<br>
<br>

- 例1  
```rb
# 数値の比較
puts 1 <=> 2 # => -1
puts 2 <=> 2 # => 0
puts 2 <=> 1 # => 1

# 文字列の比較
puts "abc" <=> "def" # => -1
puts "def" <=> "def" # => 0
puts "def" <=> "abc" # => 1

# 配列の比較
puts [1, 2] <=> [3, 4] # => -1
puts [3, 4] <=> [3, 4] # => 0
puts [3, 4] <=> [1, 2] # => 1
```
