## sortメソッドについて 
<br>

- sortメソッド 
```
配列の要素を並べ替えるためのメソッド。
昇順で並べ替える場合は単純に sort メソッドを使うが、
降順に並べ替える場合は、比較演算子をブロックで渡して並べ替え方を指定することができる。

sort!メソッド(破壊的メソッド)で元の配列をソート結果で上書きできる。
```
<br>
<br>

- 例1  
```rb
#昇順
array = [3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5]
array.sort 
#=> [1, 1, 2, 3, 3, 4, 5, 5, 5, 6, 9]
```
<br>
<br>

- 例2  
```rb
#降順1
array = [3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5]
array.sort {|a,b| b <=> a }
#=> [9, 6, 5, 5, 5, 4, 3, 3, 2, 1, 1]
```
<br>
<br>

- 例3  
```rb
#降順2
array = [3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5]
array.sort {|a,b| -a <=> -b }
#=> [9, 6, 5, 5, 5, 4, 3, 3, 2, 1, 1]
```
<br>
<br>

- 例4  
```rb
#降順3
array = [3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5]
array.sort.reverse
#=> [9, 6, 5, 5, 5, 4, 3, 3, 2, 1, 1]
```
<br>
<br>

- 例5  
```rb
#文字列
#昇順
ary1 = [ "d", "a", "e", "c", "b" ]
p ary1.sort                             
#=> ["a", "b", "c", "d", "e"]
```
