## updateメソッドについて 
<br>

- updateメソッド 
```
Hashクラスに定義されたメソッドで、Hashのキーと値を更新するために使用される。
主にRailsで使われる。

updateメソッドは、複数の引数を取ることができる。
第1引数は、更新するキーと値の組み合わせが格納されているハッシュオブジェクトである必要がある。
この引数が与えられた場合、このハッシュオブジェクトのすべてのキーと値が元のハッシュに追加され、既存のキーの値が更新さる。

第2引数以降は、ハッシュと同様に、キーと値を交互に渡すことができる。
これらの引数が与えられた場合、元のハッシュに対応するキーが存在しない場合は追加され、既存のキーの値が更新される。
```
<br>
<br>

- 例  
```rb
hash = { a: 1, b: 2 }
hash.update({ b: 3, c: 4 })
# => { :a => 1, :b => 3, :c => 4 }

hash = { a: 1, b: 2 }
hash.update(c: 3, d: 4)
# => { :a => 1, :b => 2, :c => 3, :d => 4 }
```
