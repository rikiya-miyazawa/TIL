## 条件演算子(三項演算子)  
<br>

- 条件演算子(三項演算子)  
```
条件分岐

式 ? 真だった場合の処理:偽だった場合の処理

簡潔な条件分岐は条件演算子を用いるとスッキリ書ける
複雑な条件分岐は無理せずif文を用いる
```
<br>

```rb
#if文の場合
n = 11
if n > 10
  '10より大きい'
else
  '10以下'
end

#条件演算子
n = 11
n > 10 ? '10より大きい':'10以下'

#変数に結果を代入することもできる
n = 11
message = n > 10 ? '10より大きい':'10以下'
message
#=> "10より大きい"
```
