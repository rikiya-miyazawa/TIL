## 範囲指定について 
<br>

- 範囲指定 
```
..両端を含む範囲を作成する。
...最初の値は含み、最後の値は含まない範囲を作成する。
```

<br>
<br>

- 例  
```
..
irb(main):008:1* (1..5).each do |i|
irb(main):009:1*   puts i
irb(main):010:0> end
1
2                                                                            
3                                                                            
4                                                                            
5                                                                            
=> 1..5     
```
