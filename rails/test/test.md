## test  
<br>

- test実行のコマンド  
`bin/rails test`
<br>
<br>

- より詳細なテスト結果の表示  
`bin/rails test -v`
<br>
<br>

- ディレクトリを指定してテストを実行する
`bin/rails test test/models`
<br>
<br>

- ファイル指定
`bin/rails test test/models/todo_test.rb`
<br>
<br>

- 特定の行を指定
`bin/rails test test/models/todo_test.rb:5`
<br>
<br>

- 特定のテスト名を指定
`bin/rails test test/models/todo_test.rb -n test_the_truth`
<br>
<br>
