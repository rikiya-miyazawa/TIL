## JSONとは  
<br>

- JSONとは  
```
JSON形式のデータ
{"id":1,"name":"Book 1","published_on":"2023-03-24","price":1000,"created_at":"2023-03-24T23:27:11.725Z","updated_at":"2023-03-24T23:27:11.725Z","publisher_id":1,"sales_status":null}

JSON(JavaScript Object Notation)は、データの記述や交換に使われる軽量なデータ形式の一つ。
JSONは、JavaScriptのオブジェクト表記法を基にしたテキスト形式のデータで、人間が読み書きしやすく、また様々なプログラミング言語で扱うことができる。

JSONは、キーと値のペアで構成されるオブジェクト、配列、数値、文字列、真偽値、null値などのデータ型をサポートしている。
```
