## git clone  
<br>

- git clone  
```
リモートリポジトリの内容をローカルマシンに複製するために使用されるGitコマンド
git clone <リモートリポジトリURL> <任意のローカルディレクトリ名>
git clone https://github.com/ユーザー名/リポジトリ名.git
<任意のローカルディレクトリ名>を指定することで、リポジトリを複製するディレクトリ名を指定することができる
指定しなかった場合は、リポジトリ名と同じ名前のディレクトリが生成される
```
<br>
<br>

- 主なオプション  
```
--depth: 複製する履歴の深さを指定する。履歴をすべて取得する必要がない場合に使用する。例えば、--depth 1を指定すると最新のコミットだけが複製される。
--branch: 複製するブランチを指定する。デフォルトではmasterブランチが複製される。
--bare: バックアップ用のリポジトリを作成する。ワーキングツリーを持たないリポジトリを生成するため、通常のリポジトリとは異なる。
```
